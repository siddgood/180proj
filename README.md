# Read Me

## Set up in terminal:

```sh
git clone https://github.com/siddgood/180proj

pip install virtualenv

virtualenv venv

source venv/bin/activate

pip3 install -r requirements.txt
```

## To launch:

```sh
python geo/__init__.py
```

## To view:

http://0.0.0.0:5001/
