Description:
Abstract:
This dataset contains 2019 Civic Centers, Stadiums and other large capacity facilities information for the State of Florida. This dataset contains fields denoting the physical address and contact information for civic center facilities based on data taken from the Yellow Pages, Super Pages Online, Florida Theme Parks, Florida Water Parks, Florida Amusement Parks, World Stadiums - Florida, Go Fox - Florida Venues, Google and Ticketmaster.  This data is meant to be used for planning purposes only and is not intended to represent a 100% inventory of civic centers in Florida. Please Note: Since the 2011 release this dataset's record count has been greatly minimized in order to primarily capture larger capacity facilities. Civic Center locations that have been verified are marked with the letter V in the FLAG field. This dataset is an update to the GC_CIVICCENTER_DEC11.shp layer.
Purpose:
The data was created to serve as base information for use in GIS systems for
a variety of planning and analytical purposes.
Supplemental_Information:
What is Geocoding?
Geocoding is term used to describe the act of address matching.  Geocoding
is the process of finding a geographic location (x, y point) for an address
(such as street number and name, city, state, and ZIP Code) on a map.
Geocoding is based off the typical address scheme for the US, in which one
side of the street contains even house numbers while the other side of the
street contains odd house numbers.

The geocoding process uses an algorithm to find the geographic location
of addresses. First, a street segment is identified using the zip code and
street name. Next, the geographic location of the address is matched using
the building number to determine how far down the street and on which side
of the street the building is located. Geocoding Accuracy

The locational accuracy of geocoded addresses may vary from urban to rural
areas due to the algorithm used to generate the geographic locations of
addresses. The algorithm assumes that the size of parcels are equivalent
along a road route. This assumption tends to be more consistent in urban
areas, where the size of parcels vary less than in rural areas.
Consequently, the results of geocoded addresses in urban areas are usually
more reliable than those in rural areas.

For example, the locational accuracy of rural addresses can be slightly
off because some parcels along a rural route may be 15 acres while others
may be 2.5 acres, but the geocoding algorithm assumes that the addresses
are distributed evenly along the route.

Process_Description:
GeoPlan updated the gc_civiccenter_jun08 layer using the following 
online websites (see below); 

Yellow Pages Online, Super Pages Online, 
Florida Theme Parks, Florida Water Parks, Florida Amusement Parks, 
World Stadiums - Florida, Go Fox - Florida Venues, and Ticketmaster. 

These websites provided lists of current facilities. Each record
in the layer was scrutinized and verified to be in its proper location. 
Once its location was verified appropriate attributes were updated. Facilities no 
longer deemed operational were removed from the layer. Some facilities 
that have been opened since the last update were manually added. The 
presence and location of large facilities were checked using data from CIVIC 
CENTER PARCELS IN FLORIDA and ST. JOHNS RIVER WATER MANAGEMENT
DISTRICT LAND USE AND COVER as a final verification. A new field called 
CAPACITY was added to the to show venue capacity for each record when available. 
A new field called SOURCE was added to show source information for each 
record when available. Some minor table restructuring was done to reflect the 
current table syntax used by GeoPlan.

-Total records = 2051

Process_Step:
Process_Description:
GeoPlan updated the gc_civiccenter_dec11.shp dataset in 2018-2019, the updates were 
completed on January 9th, 2019.

During this process GeoPlan used imagery, parcels, and Google Street View in 
order to verify as many facility locations as possible.

Once the facility location was verified, the point was moved to the roof of the 
building and the FLAG field was marked with a "V", non-verified facilities have been
marked with a "NV" in the FLAG field.  Facilities that are proposed to be built but 
have not yet been built are marked with a "FNV" in the FLAG field.

During the update process, several facilities were added while several facilities were
moved or deleted.

Please Note: Since the 2011 release this dataset's record count has been greatly 
minimized in order to primarily capture larger capacity facilities.

Because GC Civic Center has not been updated since 2011, it does not follow the 
current GC layer structure.  The following changes were made to update the layer to
2019 standards:

-Added and calculated the following fields: PARCELID, OWNER, DATASOURCE, MGRS, GOOGLEMAP and ACRES
-Deleted OBJECTID STATUS, SCORE, SIDE, MATCH_ADDR, AUTOID and 
USNG_FL_1K
-Renamed SOURCE as FSOURCE

Next GeoPlan Updated the following fields: LAT_DD, LONG_DD, FGDLAQDATE, UPDATE_DAY, FSOURCE.

Populated OP_CLASS and OPERATING fields based on information given in the 
NAME and OWNER fields.

Populated DESCRIPT based on NAME.

Upcased all fields and renamed dataset to GC_CIVICCENTER_JAN19.

A note about data scale: 

Scale is an important factor in data usage.  Certain scale datasets
are not suitable for some project, analysis, or modeling purposes.
Please be sure you are using the best available data. 

1:24000 scale datasets are recommended for projects that are at the
county level.
1:24000 data should NOT be used for high accuracy base mapping such
as property parcel boundaries.
1:100000 scale datasets are recommended for projects that are at the
multi-county or regional level.
1:125000 scale datasets are recommended for projects that are at the
regional or state level or larger.

Vector datasets with no defined scale or accuracy should be
considered suspect. Make sure you are familiar with your data
before using it for projects or analysis. Every effort has been
made to supply the user with data documentation. For additional
information, see the References section and the Data Source Contact
section of this documentation. For more information regarding
scale and accuracy, see our webpage at:
http://geoplan.ufl.edu/education.html
